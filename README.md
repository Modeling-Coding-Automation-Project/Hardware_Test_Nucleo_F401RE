# リアルタイム制御向けC++制御アルゴリズム STM32 Nucleo F401RE環境

## 特徴

C++制御アルゴリズムのメモリ消費、計算時間を STM32 Nucleo F401RE で検証できる環境です。

いわゆる Processor-in-the-Loop (PIL) の環境です。

## 環境構築

Visual Studio Codeをインストールし、「PlatformIO」という拡張機能をインストールしてください。
